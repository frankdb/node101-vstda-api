# Node 101 VSTDA API

This project creates a simple API that can handle GET, POST and DELETE requests.

## Getting Started

Clone repo into local directory. Run "npm start" and open on localhost:8484.

### Installing

Run "npm install" in Terminal

## Running the tests

Run "npm test"

- Node

- Express
